
## ポモトモとは？
ポモトモは、ポモドーロアプリの上にノート、友達、日課、やるべきリスト、などの機能をつけたアプリです。
使い方はとても簡単で今すぐにでも自分で試したければぜひ：
https://pomotomo.me
に行ってください
